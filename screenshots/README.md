# 截图说明

请将以下截图放在此目录:

1. `main.png` - 主界面截图(包含服务器列表和节点配置)
2. `server-list.png` - 服务器列表管理界面
3. `node-config.png` - 节点配置界面  
4. `log-viewer.png` - 日志查看器界面
5. `settings.png` - 设置和关于界面

## 截图要求

- 格式: PNG
- 推荐分辨率: 1200x800 或保持应用默认窗口大小
- 建议使用 macOS 截图工具或其他截图软件
- 确保截图内容清晰,展示主要功能

## 如何截图

1. 启动应用: `npm run tauri dev`
2. 在应用中配置好示例数据
3. 使用 `Cmd + Shift + 4` (macOS) 进行截图
4. 将截图保存到此目录并重命名
